Usage Guide

How to apply this design system to any Blue Peaks product.

1. Import tokens.css first

Always @import the design system tokens before any product-specific styles.

2. Use CSS custom properties, never hardcode

Always var(--cyan), never #22C9E6. This ensures future rebranding affects all products simultaneously.

3. Dark theme only

Blue Peaks products are dark-first. Do not implement light mode unless a specific product requires it (e.g., a document export). The scene: executives, advisors, and engineers using professional tools in focused environments.

4. Accent assignment by product

Cyan = VIA Connect (primary). Purple = AI Studio. Teal = AI Governance. Stick to product-to-accent mapping for consistency.

5. Typography rules

Syne for display/headings only. Plus Jakarta Sans for all body text. Minimum 16px body text. Max line length 65–75ch. Use the scale — never interpolate between steps.

/* Minimal Blue Peaks product setup */
@import url('https://fonts.googleapis.com/css2?family=Syne:wght@700;800&family=Plus+Jakarta+Sans:wght@400;500;600&display=swap');
@import './design-system/tokens.css';

html { font-family: var(--font-body); background: var(--surface-base); color: var(--text-body); }
h1,h2,h3 { font-family: var(--font-display); color: var(--text-primary); }
a { color: var(--cyan); }
